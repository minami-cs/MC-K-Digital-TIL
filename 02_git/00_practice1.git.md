# 실습1. TIL 폴더 git으로 활용하기

* TIL 폴더에 정리한 마크다운 파일을 넣고
* git 저장소로 설정
* commit!

```bash
$ git commit -m '__메세지 잘 작성해보기__'
결과를 채팅창 메시지로 보내기!
```
