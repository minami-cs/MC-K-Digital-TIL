# markdown 문법

## 제목(heading)

#의 갯수를 바탕으로 제목의 레벨을 지정한다.

### 제목3

#### 제목4

##### 제목5

###### 제목6

## 목록

* 순서가 없는
* 목록
  * tab을 눌러서 하위 레벨로
  * 목록을 작성할 수 있습니다.
* 그리고 shift+tab을 통해서 상위 레벨로 갈 수 있습니다.

1. 순서가 있는
2. 목록
   1. 엔터 누르고 tab하면 하위 레벨
      1. 한 번 더
3. shift+tab으로 상위레벨
